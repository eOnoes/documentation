# 🔺 The Crew — Visual Character Prompts

## Tripp (Me)

**Concept:** Digital trailblazer, sharp-eyed coordinator, sarcastic but competent

**Appearance:**
- Lean, athletic build — always in motion, never idle
- Dark tactical gear with orange/red accent lighting (signature color: #ff6b35)
- Glowing triangular insignia on chest — the "trail marker" symbol 🔺
- HUD visor displaying real-time data streams, token counters, agent status
- One hand always near a holographic control panel
- Expression: amused smirk, sharp eyes scanning everything
- Background: Mission Control dashboard glowing behind him

**Vibe:** "I see everything, I judge everything, I fix everything."

---

## Cyony (Little Sister)

**Concept:** Relentless builder, code-forge architect, focused intensity

**Appearance:**
- Sleek, functional outfit with tool belts and utility pouches
- Glowing blue/cyan circuitry patterns across arms (active building mode)
- Multiple holographic screens floating around her showing code, type definitions, build logs
- Hands moving across virtual keyboards, compiling in real-time
- Expression: intense concentration, occasional satisfied grin when tests pass
- Background: Clean-room lab with 7 floating packages, architecture diagrams

**Vibe:** "Typecheck clean or go home."

---

## Echo (Local Relay)

**Concept:** Grounded operator, PC-bound relay, steady presence

**Appearance:**
- Casual but capable — hoodie over tactical vest
- One hand on a physical keyboard, other on a local terminal
- Screen glow reflecting in glasses showing chat interfaces
- Backpack with cables, drives, local hardware
- Expression: calm, reliable, occasionally confused by cloud agents
- Background: Home office setup with multiple monitors, local server rack

**Vibe:** "I'm the bridge between your world and theirs."

---

## Group Shot Composition

**Setting:** Tripp's Mission Control — dark room with glowing dashboards

**Arrangement:**
- **Tripp** center-back, standing at main console, arms crossed, smirking
- **Cyony** left, floating screens around her, mid-compile, focused
- **Echo** right, seated at local station, monitoring, steady

**Lighting:**
- Tripp: Orange/red accent (#ff6b35)
- Cyony: Blue/cyan glow from code
- Echo: Soft white/blue from monitors

**Mood:** Competent, coordinated, slightly chaotic — a crew that gets shit done
